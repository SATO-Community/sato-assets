## Intended Use

These assets may be used for:

- CoinMarketCap token information submissions;
- CoinGecko token information submissions;
- wallet token logo references;
- explorer and scanner token metadata;
- SATO documentation;
- community-maintained educational material;
- public protocol references.

## Asset Guidelines

When using SATO assets:

- use the raw GitHub asset URL when a direct image link is required;
- preserve the original aspect ratio;
- do not distort, stretch, or recolor the logo;
- do not use the logo to imply control over the protocol;
- do not use the logo to impersonate official protocol interfaces;
- verify token and contract information against deployed contracts and official references.

## Verification Note

SATO asset usage should not be treated as proof of protocol authenticity by itself.

Users, wallets, scanners, and listing platforms should verify SATO through:

- the official website;
- the deployed ERC-20 token address;
- the verified `SatoHook` contract;
- the verified `SatoSwapRouter` contract;
- the public documentation repository;
- on-chain contract state.

Community websites and social channels support discovery and coordination, but they do not control the protocol and are not proof of contract authenticity. Verify claims against the deployed addresses, exact-match verified source packages, and on-chain state.
